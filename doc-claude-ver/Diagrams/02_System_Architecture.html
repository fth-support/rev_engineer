Main Processing Flow (otmForm_Timer — every 500ms)

1
Pre-check: Disable timer, verify Log file, read Registry (SentSale, Online status)
2
Connect: W_DATbConnectOrChk() to Central Server + health check (SELECT FTTmnCode FROM TCNMTerminalMtn)
3
Store Check: W_GETbStoreOpen() verifies TsysTerminalSta.FTTmsOpnStore = 'Y'
4
Config: Fetch TSysSync table list, Terminal Number, Sale Date
5
Sync Loop: For each table in TSysSync → fetch records where FTStaSentOnOff='0' → INSERT/UPDATE to Server with tokenization → mark as '1'
6
Points: W_UPDxUpdatePoint() — SUM points per member, tokenize member ID, UPDATE/INSERT TCNMMallCard, log to TPSTBPHis
7
Cleanup: Save Registry state, re-enable timer
