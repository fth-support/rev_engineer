Member Points Processing Flow

ServiceTransfer — W_UPDxUpdatePoint() • การประมวลผลคะแนนสะสมสมาชิกไปยัง Member DB

1
Fetch pending transactions — ดึงธุรกรรมที่มีคะแนนค้าง
SELECT DISTINCT FTTmnNum, FTShdTransNo, FTShdTransType, FDShdTransDate FROM TPSTSalePoint WITH(NOLOCK) WHERE ISNULL(FTRemark,'')='0' AND ISNULL(FTSpoMemID,'')<>'' ORDER BY FTShdTransNo
▼
2
Sum points per member — รวมคะแนนต่อสมาชิก
SUM(FCSpoPoint) GROUP BY FTSpoMemID, FDDateUpd, FTWhoUpd — โดย FDDateUpd เก็บ ExpDate (YYMM) และ FTWhoUpd เก็บ Flag ChkExp (Y/N)
▼
3
Tokenize member ID — แปลงรหัสสมาชิกเป็น Token
SP_PRCbToken("TPSTSalePoint", MemID, …, pbNoSearchDB=True) → บันทึก Mapping ลง TPSTTokenLst บน Member DB (Token ไม่สำเร็จ = ข้ามสมาชิกรายนั้น)
▼
4
Apply sign by TransType — กำหนดเครื่องหมายคะแนน
+ Point: Sale (03), Deposit (06), VIPSale (10), DutyFreeSale (13), Voucher (15), AddDeposit (19), SaleTaxInc (38), SaleManualTax (43)
− Point: Return (04), IPV (05), Redeem (07), DepositCancel (16), AddDepositCancel (20), CancelVoucher (21), IPV* (26-28, 32), ReturnTaxInc (39), ReturnManualTax (44)
▼
5
Update / Insert member card — ค้นหาและปรับปรุงบัตรสมาชิก
ChkExp = 'Y' → ค้นด้วย CardCode + SUBSTRING(ExpDate,1,4) = YYMM • อื่น ๆ → ใบล่าสุด ORDER BY FDMcdExpDate DESC
พบสมาชิก → UPDATE
FCEarned += pt
FCBalance = FCMcdTotalPoint + FCEarned + pt
FTWhoUpd = 'ServiceTranfer'
ไม่พบ → INSERT 'New Member'
FTMcdStaAct='A', FCEarned = FCBalance = pt
FDMcdExpDate = วันสุดท้ายของเดือน YYMM
▼
6
Write history + mark done — บันทึกประวัติและปิดงาน
INSERT INTO TPSTBPHis(Keys, MemberID, CardPoint, TotalBP, BalanceBP, …) → UPDATE TPSTSalePoint SET FTRemark='1', FDDateUpd = McdExpDate
On error: ตรวจสอบ Web Service + Member DB Connection ใหม่ → Resume Next (ข้ามไปเรคอร์ดถัดไป ไม่หยุดทั้งรอบ)
