Tokenization System

ServiceTransfer — SP_PRCbToken() / SP_PRCbDeToken() / SP_CHKbWebservice()

Token Flow — SP_PRCbToken()
Guard 1: bVB_AlwToken = False OR Len(CardNo) < 7 → Return original value (no token)
▼ passes
Guard 2: Len(CardNo) < 14 OR CardNo = "" → Return original value (no token)
▼ passes
DB Search: SELECT TOP 1 FTToken FROM TPSTTokenLst WHERE FTValue=CardNo AND FTValue<>FTToken ORDER BY FDDateIns DESC
Found → Return existing token from DB (skip web service call)
▼ not found
SOAP Call: SP_CHKbWebservice() → Create MSSOAP.SoapClient30 → MSSoapInit(URL) → Set Timeout 10,000ms
▼ ready
InsertToken: oToken.InsertToken(NAEUsr, NAEPwd, DBUsr, DBPwd, DB, CardNo, FIRST_SIX_TOKEN, LuhnCheck=0)
Token received → Return new token
Error or empty → Return False (token failed, Error 557)
